Hexo相关说明

2022/12/6 hexo

window 相关指令

创建文件夹  md 文件夹名
创建新文件 type nul >文件名
创建有内容文件 echo "123" >文件名

查看文件内容 type 文件名
查看文件目录及子文件 dir

删除空文件夹 rd 文件夹名
删除文件(/删除文件夹) del 文件名

hexo blog 搭建

node -v
npm -v
npm install -g hexo-cli
hexo -v
hexo init myblog
cd myblog
hexo server // 启动本地服务器
http://localhost:4000/ // 本地访问

github 创建新仓库

注意仓库名 需要是自己 github 账户名 (eg:账户名为 gyy1129 -> gyy1129.github.io)

安装部署插件

npm install --save hexo-deployer-git

配置 _config.yml 文件

deploy:
  type: 'git'
  repo: https://github.com/gyy1129/gyy1129.github.io.git
  branch: master

hexo deploy // Deploy to remote sites

创建新页面

hexo new "first passage"
hexo clean // 清理
hexo generate // Generate static files
